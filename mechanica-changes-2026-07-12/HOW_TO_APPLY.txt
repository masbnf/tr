این زیپ شامل تمام فایل‌های جدید/تغییرکرده‌ی امروز است (۳۱ فایل)،
دقیقاً با همان مسیر نسبی که در ریپو دارند.

نحوه‌ی اعمال:

1. محتوای این زیپ را استخراج کن و مستقیم روی پوشه‌ی ریپوی محلی‌ات
   (همان جایی که package.json وجود دارد) کپی/جایگزین کن.
   یعنی پوشه‌ی docs/ و src/ داخل زیپ باید با docs/ و src/ داخل
   ریپوی تو ادغام شوند (فایل‌های هم‌نام جایگزین، بقیه اضافه می‌شوند).

2. چون package.json تغییر کرده (آپدیت next):
   npm install

3. تست محلی قبل از commit:
   npm run typecheck
   npm run lint
   npm run build

4. اگر همه چیز پاک بود:
   git add -A
   git commit -m "docs + security patch + request form + admin panel"
   git push
